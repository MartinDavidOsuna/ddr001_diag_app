# DIAGNOSTICO HIDRANTES

Aplicación Flutter offline-first para inspección técnica del Distrito de Riego 001. Gestiona REPORTE VISUAL (RV), REPORTE FUNCIONAL (RF), evidencia fotográfica, recuperación local, trazabilidad y sincronización local preparada para una API futura.

La documentación técnica y las matrices manuales se encuentran en `docs/`; los planes aprobados se conservan en `plans/`.
